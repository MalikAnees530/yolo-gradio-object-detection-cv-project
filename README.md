# Real‑Time YOLO‑Based Object Detection System with Interactive Gradio Web Interface

This repository contains the code for our Computer Vision course project.  
The project implements a real‑time object detection module using the YOLOv8 model and an interactive Gradio web interface, running in Google Colab.

## Project Overview

- Task: CV‑based module (object detection)  
- Model: Ultralytics YOLOv8n (pre‑trained on COCO)  
- Frontend: Gradio web UI  
- Environment: Google Colab, Python 3
